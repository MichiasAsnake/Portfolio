Blog It


Blog it is a site I created with the purpose of practicing some Javascript methods. The design inspiration comes from both Twitter and Reddit, mainly from the way they structure their posts. During this project I learned alot about Javascript concepts as well as CSS styling.


Features

• Create new posts and add them to the feed.

• Up and down vote buttons that increment like count.

• Comments drop down section that displays messages from other users.

• Fully scales to display on desktop and mobile devices
