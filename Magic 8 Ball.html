Magic 8 Ball


This site takes the classic Magic 8 Ball game and puts it into a fun to play website. Just like the real game, users are generated random responses after every input has been entered. I utilized CSS animations in this website to create a shaking effect that mimicks the real Magic 8 Ball.


Features

• Generate a random response based on the phrases from the original Magic 8 Ball.

• 8 Ball shakes between inputs.

• Fade animations placed on responses.
