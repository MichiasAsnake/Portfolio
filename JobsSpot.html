JobsSpot


JobsSpot was a project I created to search job listings from multiple websites. This project is entirely created in React using CSS and some HTML as well. JobsSpot utilizes the JSearch API to aggregate data form job sites worldwide.


Features

• Search listings from multiple job websites.

• Apply directly to positions on job publisher's websites.

• See when a job was posted.
