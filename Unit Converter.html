Unit Converter


Here is a simple unit converter I created that converts any numerical input from metric into imperial units. This project utilizes Javascript to create the conversion as well as minimal HTML and CSS styling.


Features

• Converts from both metric and imperial units.

• Includes length, volume, and mass measurements.

• Utilizes Javascript, HTML, and CSS.
