Coolors Site Clone


This site is a clone of the popular color scheme generator website, coolors.com. For this clone I really focused on recreating the all UI elements and images of the website. For this project I utilized Adobe Illustrator, Photoshop, HTML, CSS, Figma and Javascript. To generate the color's, hex numbers, and color schemes, I utilized the color api which gave me access to all of this data and more.


Features

• Generate random unique color scheme's.

• Create a scheme based off of a color you chose in the color picker, or randomize your color scheme.

• Choose from a list of 7 different color schemes.

• Use the spacebar to quickly generate new colors on every click.

• Users are provided a hex color for every color to easily recreate color palettes.
